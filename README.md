# plg_content_google_analytics
Плагин для google analytics

Атребуты тегов 
  + **data-conversions-ec-list="Category Page"**  -  ( Список продуктов  Значение == название списка в Google Analytics ) 
    
    + **data-conversions-ec-product="135"** ( Товар в блоке Значение = ID товара. Является Маркером блока товара  )
    
        + **data-conversions-ec-product-name** 
        ```markdown
        Название товара в блоке 
        Значение = Название товара либо Тег 
        должен содержать название товара
        ```
                
        + **data-conversions-ec-product-price** 
        ```markdown
        Цена товара в блоке 
        Значение = Цена либо Тег 
        должен содержать цена товара
        ```
        
        
        + **data-conversions-ec-product-category="Люстры"** 
        
        ( Категория товара Значение == Название категории. )
